# SuburbRank Python App

Fast-launch MVP stack:
- FastAPI
- Jinja2 templates
- SQLAlchemy
- PostgreSQL (SQLite fallback for local dev)
- Stripe
- WeasyPrint or similar for PDF reports later

Goal:
Launch a functional subscription app as fast as possible with:
- landing page
- pricing page
- login/register
- dashboard
- suburb rankings
- suburb profiles
- weekly reports
- admin fake-data generation and scoring
