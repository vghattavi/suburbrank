# UI Spec

## Design direction
- modern startup feel
- clean spacing
- investor credibility without traditional finance stiffness
- card-based layout
- responsive table design

## Main views
### Dashboard
- hero summary
- top 10 suburbs section
- featured suburb card
- latest report card
- quick stats cards

### Suburbs list
- ranked table
- score pill
- summary line
- search and sort controls

### Suburb detail
- suburb header with score
- metrics grid
- score breakdown visual/card list
- explanation section
- risk flags section

### Weekly report
- report header
- methodology note
- ranked top 10
- notable movers / featured suburb
- download CTA

## Public site
### Home page
- hero

- sample ranking preview
- CTA to subscribe

### Pricing page
- single $20/mo tier
