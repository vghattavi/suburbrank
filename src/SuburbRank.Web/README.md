# SuburbRank.Web

This folder is the intended Blazor web application root.

## Intended stack
- .NET 8
- Blazor Server or Blazor Web App (server-first)
- ASP.NET Core Identity
- EF Core
- PostgreSQL
- Stripe
- Azure deployment

## Current status
The dotnet SDK is not available in the current workspace runtime, so the project cannot be scaffolded or compiled from here yet.

To keep momentum, the implementation blueprint and starter source files are being created manually so the real app can be scaffolded against them as soon as the SDK is available.
