# SuburbRank

Modern startup-style SaaS for Australian retail property investors.

## Positioning
SuburbRank is a suburb-ranking platform focused on NSW for MVP. It gives users:
- a dashboard with ranked suburbs
- individual suburb profiles
- a weekly report

Initial data and scoring are auto-generated/fake for rapid MVP development, with a path to replace them with real datasets later.

## MVP Goals
- Public marketing site
- User registration/login
- Subscription paywall
- Dashboard with ranked suburbs
- Individual suburb profile pages
- Weekly report view + downloadable report
- Admin tools to auto-generate data and scoring runs

## Product Feel
- modern startup aesthetic
- investor-oriented but clean and approachable
- fast, polished, minimal

## Current Decisions
- Target users: retail property investors
- Geography: NSW only for MVP
- Product positioning: suburb ranking
- Experience: dashboard + weekly report
- User-facing content: top-ranked suburbs + suburb profiles
- Scoring/data: auto-generated initially
- Stack direction: Python / FastAPI / SQLAlchemy / Stripe / Render
- Brand/domain: SuburbRank / suburbrank.com.au
